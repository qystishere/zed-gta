# zed-gta

Syntax highlighting for GTA San Andreas data files in [Zed](https://zed.dev).

## Supported formats

| Format  | Extension | Description                                         |
| ------- | --------- | --------------------------------------------------- |
| GTA IDE | `.ide`    | Item definitions (objects, vehicles, peds, effects) |
| GTA IPL | `.ipl`    | Item placements (instances, zones, garages, jumps)  |
| GTA DAT | `.dat`    | Data loading directives (gta.dat, default.dat)      |
